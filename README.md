#SubtitleRenamer v1.3.1

####동영상파일에 맞춰 자막파일의 이름을 변경해주는 프로그램 입니다

**사용법**

1. 원하는 동영상을 왼쪽 리스트박스에 드래그 앤 드롭.
2. 변환하고싶은 자막을 오른쪽 리스트 박스에 드래그 앤 드롭.
3. 변환
